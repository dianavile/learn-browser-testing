## To start

1. Make a free [percy.io](https://percy.io/) account.
2. In the Percy dashboard, make a new project and copy the token from project settings
3. Copy `.env.sample` over to `.env` and fill in your token

## To run

```sh
npm start

# or

yarn start
```
